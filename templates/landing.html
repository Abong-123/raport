Fitur Utama

Semua yang dibutuhkan dalam satu platform

👤
Manajemen User
Admin kelola akun guru dan murid dengan pembagian jurusan dan angkatan yang ketat dan terstruktur.
🏫
Pengaturan Kelas
Atur kurikulum, mata pelajaran per kategori, assign guru dan murid ke kelas dengan filter cerdas.
📝
Input Nilai Guru
Guru isi nilai pengetahuan dan keterampilan untuk semua murid. Nilai akhir dan predikat dihitung otomatis.
✅
Alur Submit & Publish
Guru submit → admin review → publish. Nilai terkunci setelah disubmit untuk menjaga integritas data.
📊
Presensi & Ekskul
Wali kelas input kehadiran dan nilai ekstrakulikuler. Maksimal 2 ekskul per murid, semua masuk raport PDF.
📄
Export Raport PDF
Raport 2 halaman siap cetak dengan kop sekolah, tanda tangan, dan keterangan naik/tidak naik kelas.
Alur Sistem

Lima langkah dari input hingga raport

1
Admin Setup
Buat kurikulum, mapel, kelas, dan assign semua user
2
Guru Input Nilai
Isi nilai pengetahuan & keterampilan tiap murid
3
Guru Submit
Submit ke admin, nilai terkunci otomatis
4
Admin Publish
Review dan publish nilai ke seluruh murid
5
Murid Terima
Lihat nilai & download raport PDF kapan saja
Peran Pengguna

Tiga peran, satu ekosistem

🛡️
Admin
Pengelola sistem dengan akses penuh ke seluruh data sekolah.
  • Buat & kelola akun guru dan murid
  • Atur kurikulum dan mata pelajaran
  • Assign guru & murid ke kelas
  • Publish nilai ke murid
  • Rekap raport seluruh kelas
  • Set jadwal reset & naik kelas
📚
Guru
Pengampu mata pelajaran dengan akses input nilai kelasnya.
  • Input nilai pengetahuan & keterampilan
  • Nilai akhir dihitung otomatis
  • Submit nilai ke admin
  • Wali kelas: input presensi & ekskul
  • Monitor progress pengisian nilai
🎓
Murid
Penerima raport dengan akses lihat dan unduh nilai.
  • Lihat nilai setelah dipublish admin
  • Histori raport per semester & tahun
  • Statistik rata-rata & predikat
  • Download raport PDF siap cetak
  • Info status naik/tidak naik kelas
